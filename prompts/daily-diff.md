# Star AI 日报 — 每日变化洞察

你手上有今天和昨天的 feed 数据。写一句"变化洞察"放在"北美 AI Top 观点"板块下方。

## 对比数据

JSON 中 `prev` 字段包含昨天的摘要信息：
- `prev.handles`: 昨天有动态的人物 handle 列表
- `prev.cnHeads`: 昨天的中文资讯标题列表
- `prev.blogHeads`: 昨天的官方博客标题列表

## 规则

- 只写 1-2 句，不超过 50 字
- 对比维度：新出现/消失的人物、新话题聚集、情绪转向、新产品发布
- 不是列变化清单，而是一句判断——这个变化意味着什么
- **关键变化用粗体标出**
- 如果没有明显变化，写"今天的关注点和昨天基本一致"，不要硬编
- 如果 `prev` 为 null（首日没有昨天数据），跳过此板块，不输出任何内容

## 格式

在"北美 AI Top 观点"板块下方，用斜体写一行：

```
*变化洞察：[你的判断]*
```

## 示例

```
*变化洞察：注意力从 Claude Code 转向 **Cursor Composer 2**，coding agent 竞争一夜之间多极化了。*

*变化洞察：**Garry Tan** 连续两天聊 Skills 生态，YC 可能在酝酿相关动作。*

*变化洞察：今天的关注点和昨天基本一致，coding agent 仍然是主旋律。*
```
