🎙️ AI 超一线播客

**Latent Space — "Why Agents Keep Failing (And How to Fix Them)"**

一句话核心：大多数 agent 的失败不是智能问题，是 **工具调用问题**。

嘉宾是 LangChain 的 Harrison Chase，作为 agent 框架最大的赌注之一的掌舵人，他对"为什么 agent 还没起飞"的反思比任何人都切身。核心发现：**当 agent 可用的工具超过 15 个时，工具选择准确率从 95% 暴跌到 60%**。解法不是更聪明的模型，而是按任务精选工具集。

另一个值得注意的趋势："Eval 驱动开发"正在取代靠感觉调 prompt。如果你不量化评估，就是在瞎猜。

> "The model is smart enough. The problem is we're giving it a toolbox the size of a hardware store and asking it to pick the right screwdriver."
