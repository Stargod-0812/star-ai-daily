# ⭐ Star AI 日报

**by starrliao | AI 驱动的前沿信息精选**

Star AI 日报每天从 X/Twitter、YouTube 播客、OpenAI/Google/HuggingFace 官方博客、36氪等全球最前沿的 AI 阵地，精挑细选最有价值的信息，用最地道的中文讲给你听。

**关注世界最前沿 AI 阵地，不只 FOMO 跟风。**

## 你会得到什么

- 🎙️ 顶级 AI 播客新节目的精华摘要
- 🐦 29 位 AI 领域关键人物在 X/Twitter 上的核心洞察
- 🇨🇳 国内 AI 一线动态（融资、产品、政策）
- 🌐 OpenAI / Google / HuggingFace 官方博客精选
- 🔗 所有原始内容的链接
- 🌏 中文、英文或双语版本
- 📄 每期日报自动生成精美 HTML 网页存档

## 快速开始

1. 在你的 AI agent 中安装此 skill（WorkBuddy、OpenClaw 或 Claude Code）
2. 输入 "设置 Star AI 日报" 或执行 `/ai`
3. Star 会以对话方式引导你完成设置
